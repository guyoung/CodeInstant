CodeInstant
===================================

CodeInstant是一款前端代码快速搭建静态展示网站工具。

![CodeInstant](doc/images/code_instant_01.png)

    |- build                    脚本文件夹
    |--- build.js
    |- data                     数据文件夹
    |--- codes                  代码数据文件夹
    |----- codes_3

    |----- vue
    |------- vue_quick_strat

    |--------- main.js
    |--------- style.css
    |------- vue_route
    |--------- index.html
    |------ code.config.json    
    |--- home
    |----- index.md
    |--- pages                  文档数据文件夹                 
    |----- page_1
    |------- index.md
    |----- page_2
    |------- index.md
    |----- page_3
    |------- index.md
    |------ page.config.json
    |--- app.config.json
    |- framwrok                 框架文件夹
    |-index.html                网站首页
